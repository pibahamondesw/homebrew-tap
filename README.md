# pibahamondesw/homebrew-tap

Homebrew tap for [WorktreeManager](https://github.com/pibahamondesw/WorktreeManager)
and other apps.

## Install

```sh
brew install pibahamondesw/tap/worktreemanager
```

The app auto-updates itself after install. The cask here is bumped automatically
by WorktreeManager's release workflow on each tagged release.
